# DESIGN SYSTEM

## Brand Identity

**Name:** StoryForge
**Tagline:** "Your voice. Your story. Your book. Finally finished."

**Color Palette:**
```
Primary:      #2563EB (Royal Blue) - CTAs, links, progress
Secondary:    #7C3AED (Purple) - AI features, Source Brain
Success:      #10B981 (Emerald) - Complete, ready states
Warning:      #F59E0B (Amber) - Partial, attention needed
Error:        #EF4444 (Red) - Failed, urgent
Neutral:      #1F2937 (Dark Gray) - Text
Background:   #F9FAFB (Light Gray) - Page background
Surface:      #FFFFFF (White) - Cards, modals
```

**Typography:**
```
Headings:     Inter (Bold/Semibold)
Body:         Inter (Regular/Medium)
Monospace:    JetBrains Mono (code, transcripts)
```

**Spacing Scale:** 4px base (4, 8, 12, 16, 24, 32, 48, 64)

**Border Radius:** 
- Small: 4px (buttons, inputs)
- Medium: 8px (cards)
- Large: 12px (modals, panels)

# INTERACTION PATTERNS

## Loading States
- Skeleton loaders for content areas
- Spinner with text for AI processing
- Progress bars with percentage for long operations

## Success States
- Green checkmark animation
- Confetti for milestones (chapter complete, 50%, etc.)
- Toast notifications for quick confirmations

## Error States
- Red highlight on failed fields
- Retry buttons for API failures
- Helpful error messages (not technical jargon)

## Empty States
- Friendly illustrations
- Clear CTAs ("Add your first question", "Upload a source")
- Helpful tips

---

# ANIMATION & TRANSITIONS

## Page Transitions
- Fade in/out (200ms)
- Slide from right for detail views

## Micro-interactions
- Button hover: subtle scale (1.02) + shadow
- Card hover: slight lift + shadow
- Recording button: pulse animation when active
- Progress bars: smooth fill animation

## Celebratory Moments
- Chapter completion: confetti burst
- 50% milestone: achievement badge popup
